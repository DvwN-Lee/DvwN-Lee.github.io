# 포트폴리오 사이트

이동주 Cloud/DevOps Engineer & Backend Developer 포트폴리오 사이트입니다.

## 특징

- **모던한 디자인**: 다크 테마 기반의 Interactive UI/UX
- **반응형 웹**: 모바일, 태블릿, 데스크톱 모든 환경 지원
- **인터랙티브 요소**: 스크롤 애니메이션, 파티클 효과, 동적 콘텐츠
- **프로젝트 필터링**: 카테고리별 프로젝트 분류 및 필터
- **상세 모달**: 각 프로젝트의 상세 정보 모달 제공

##  기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 커스텀 속성, 그리드, 플렉스박스, 애니메이션
- **Vanilla JavaScript**: 프레임워크 없는 자바스크립트
- **외부 라이브러리**:
  - Particles.js: 배경 파티클 효과
  - AOS (Animate On Scroll): 스크롤 애니메이션
  - Font Awesome: 아이콘
